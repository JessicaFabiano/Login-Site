Projeto: Página de Login com vídeo de fundo

Breve descrição
----------------
Página de login estática com vídeo de fundo. Código limpo e acessível: formulário com label, estilo responsivo e fallback para vídeo. HTML5 + CSS3. 

Tecnologias utilizadas
------------------
- HTML5
- CSS3
- Google Fonts (Nunito)
- Arquivos estáticos: `index.html`, `style.css`, `1.png`, `2.mp4` (vídeo)

Como ver o projeto localmente
----------------------------
1. Abra a pasta do projeto no seu sistema de arquivos.
2. Clique duas vezes em `index.html` ou abra-o no navegador (Chrome/Edge/Firefox).
